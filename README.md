
## Description

Notes app

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running the swagger (APIS documentation)

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
- Swagger url : http://localhost:3000/api
